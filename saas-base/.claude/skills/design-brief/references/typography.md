## Guia de decisão rápida

| Produto/Mood | Pairing recomendado |
|--------------|---------------------|
| SaaS B2B genérico | Friendly SaaS (Plus Jakarta Sans) |
| SaaS premium/luxo | Premium Sans (Satoshi) |
| Dashboard data-heavy | Minimal Swiss (Inter/Inter) ou Financial Trust |
| Fintech/banking | Financial Trust (IBM Plex Sans) |
| Healthcare | Medical Clean (Figtree + Noto Sans) |
| Startup bold | Startup Bold (Clash Display + Satoshi) |
| Gaming | Gaming Bold (Russo One + Chakra Petch) |
| Luxury/moda | Classic Elegant (Playfair + Inter) |
| Dev tools | Developer Mono (JetBrains Mono + IBM Plex) |
| Criativo/portfólio | Geometric Modern ou Fashion Forward |
| Kids | Playful Creative (Fredoka + Nunito) |
| Wellness | Wellness Calm (Lora + Raleway) |
| Web3/crypto | Crypto/Web3 (Orbitron + Exo 2) |
| Editorial | Editorial Classic ou Magazine Style |
