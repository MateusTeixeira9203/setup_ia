## Guia de decisão rápida

| Pedido | Estilo recomendado |
|--------|--------------------|
| "moderno, limpo, SaaS" | Minimalism & Swiss |
| "premium, glass, moderno" | Glassmorphism |
| "dark, tech, dashboard" | Dark Mode OLED + Data-Dense |
| "bold, startup, diferente" | Vibrant & Block-based ou Neubrutalism |
| "infantil, amigável" | Claymorphism |
| "Apple-style, cards" | Bento Box Grid ou Bento Grids |
| "AI product, chatbot" | AI-Native UI |
| "portfolio criativo" | Exaggerated Minimalism ou Anti-Polish |
| "ecommerce luxo" | E-commerce Luxury + Dimensional Layering |
| "landing page que converte" | Conversion-Optimized ou Hero-Centric |
| "gaming, neon" | Cyberpunk UI ou Retro-Futurism |
| "wellness, natureza" | Organic Biophilic ou Nature Distilled |
